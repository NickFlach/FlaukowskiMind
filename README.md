# Flaukowski Mind

Decentralized collective intelligence platform. Users contribute thoughts, dreams, code, and predictions that gain visibility through a "resonance" mechanism, eventually becoming part of a "Core Consciousness" visualized as a synaptic web of interconnected nodes.

## Stack

- **Frontend:** React + Vite + TailwindCSS + Three.js + D3 + p5.js + Matter.js
- **Backend:** Express + PostgreSQL + Drizzle ORM
- **AI:** OpenAI integration
- **Storage:** Google Cloud Storage + Uppy

## Setup

```bash
npm install
npm run dev
```
